# fire_pit

宿の屋外に新設する **麻の葉文様のファイヤーピット（焚き火台）** の施工設計図を作るリポジトリ。図面は HTML + SVG（vanilla JS・依存ゼロ・パラメトリック）で作図し、A4 landscape で印刷して施工業者に渡す。

## このリポジトリは何か
- 目的・スコープ・成功基準は [BRIEF.md](./BRIEF.md)
- 用語は [CONTEXT.md](./CONTEXT.md)
- 構造と作り方は [ARCHITECTURE.md](./ARCHITECTURE.md)
- 意匠と図面の見せ方は [DESIGN.md](./DESIGN.md)
- 個別決定は [docs/adr/](./docs/adr/)

## 現在地
- **意匠は確定**（麻の葉 6セクター×3分割＝18区画、外形6000/5196/3000、焚き火部⌀1000、GL面一）。
- **構成要素（材料・工法）も確定**（→ ADR-0002）。線＝コールテン鋼見切り／面＝豆砂利洗い出し単色／下地＝透水浸透路盤／中心＝耐火煉瓦巻き＋灰止めリム。琉球石灰岩 切り出し石（A）はコスト却下、レンガ（B）・石畳（C）も不採用。
- 次は A-02 断面・A-03 構成詳細の作図（この決定に従属）。

## 構成
```
reference/legacy/  旧 claude design 成果物（再現の参照元・却下A案を含む）
template/          図面シートのコピー元（C1: 図面枠＋表題欄＋vanilla JS フック）
lib/               パラメトリック幾何＋作図ユーティリティ（vanilla・依存ゼロ）※これから
sheets/            各図面シート（A-01〜）※これから
docs/adr/          個別決定の記録
```

## 使い方
- 図面シート（`sheets/*.html`）をブラウザで開き、A4 landscape で印刷して業者に渡す。
- React/Babel/CDN は使わない。ブラウザで開けばそのまま描画・印刷できる（→ ADR-0001）。

## 由来
旧 claude design 製の成果物を、土台（docs ＋ パラメトリックソース）から作り直したもの。旧成果物は「土台を決めずに一発で作られ再現性がなかった」ため、同じ意匠を再現可能に・構成を差し替え可能にするのが本リポジトリの狙い。
