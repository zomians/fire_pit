# CONTEXT — 用語

この案件で使う言葉の正準定義。図面・docs・コミットで言葉が揺れたらここに合わせる。
**意匠の用語**と**構成要素の用語**を分ける（いずれも確定 → 構成は ADR-0002・芝生目地案）。

## 意匠・幾何（確定）
- **ファイヤーピット（焚き火台）** … 宿の屋外に新設する、麻の葉文様に割り付けた焚き火設備。本案件の製作対象。
- **麻の葉文様** … 正六角形を中心から6つの正三角形（セクター）に分け、各セクターを重心スポークで3分割した割付。計18区画で六角形が埋まる。
- **基準六角形** … 全高6000mm に内接する正六角形（R=中心〜頂点=3000、一辺3000、point-up）。全割付の基準＝理論線。
- **セクター** … 中心から見た6分割の1つ（正三角形）。番号 S1〜S6。
- **区画** … 1セクターを重心スポークで3分割した二等辺三角形（約 3000 × 1732 × 1732mm）。命名 `S<セクター>-<A/B/C>`（例: S1-A）。全18区画。意匠上の割付単位であり、**物理的な部材割りとは独立**（区画＝洗い出しの島、境界＝芝生帯 → ADR-0002・芝生目地案）。
- **重心スポーク** … 各セクター（正三角形）の重心から3頂点へ引く3本の線。これで1セクターが3区画に分かれる。正三角形なので重心＝外接円中心、重心から各頂点まで等距離（約1732mm = 辺/√3）。
- **焚き火部** … 中央にくり抜く小六角形の開口（⌀1000 / 半径500）。薪直火の火床（→ 灰止めリム）。
- **GL±0（面一）** … 天端を周囲の既存芝生面と同じ高さ（グランドレベル）に揃える納まり。**中心火床のみ立ち上がり例外**。

## 構成要素（確定 — ADR-0002・芝生目地案）
- **構成要素** … 18区画の意匠を物理的に成立させる材料・工法。**確定**（→ ADR-0002・芝生目地案）。麻の葉の線を芝生（負の空間）で出し、区画は舗装の島とする。
- **芝生帯（線）** … 麻の葉の線（18区画境界）を出す幅 ≈120mm の芝生の帯。外周の既存芝生と連続する。
- **区画島** … 芝生帯に囲まれた舗装の島。中身は豆砂利洗い出し・単色。中心に向かう島は火床の耐火煉瓦リムで止まる。
- **下地** … 区画島＝砕石路盤＋透水路盤＋透水洗い出し。芝生帯＝客土＋芝。全面 GL±0・全面浸透。
- **火床構成** … 中心⌀1000 の内周を巻く耐火煉瓦＋天端から立ち上げた灰止めリム＋底の砂利排水層。
- **灰止めリム** … 中心火床を天端から1〜2段（≈65〜130mm）立ち上げた縁。灰の外周への飛散を止める。
- **エッジ材** … 区画島と芝生帯の境を縁切りする土留め見切り。
- ※「切り出し / 敷き砂」等は旧成果物（却下A案）の語であり、本構成の正準用語ではない。
