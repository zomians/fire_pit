# BRIEF — 麻の葉 ファイヤーピット 施工設計図

## 何を
宿の屋外に新設する **麻の葉文様のファイヤーピット（焚き火台）** の施工設計図を作る。図面は HTML + SVG で作図し、印刷用シート（A4 landscape）として出力して施工業者に渡す。

## なぜ
宿の屋外設備をアップデートするため、自分で意匠・寸法を決めたファイヤーピットを製作・設置する。図面は構成材の製作/調達発注と現場設置の根拠になる。

## 確定していること
ここが本案件の現在地。**意匠・外形に加え、構成要素（材料・工法）も確定**（→ ADR-0002・石畳案）。

### 確定（意匠・外形）
- 外形: 正六角形（point-up）、全高 6000 / 全幅 約5196 / 一辺 3000mm
- 意匠: **麻の葉文様 = 6セクター × 3分割 = 計18区画**（割付の幾何は DESIGN.md で確定）
- 中央: 焚き火部 ⌀1000（小六角形くり抜き）
- 納まり: 天端は周囲の既存芝生面と面一（GL±0）。**ただし中心火床のみ灰止めのため立ち上げる例外**。

### 確定（構成要素・工法）— ADR-0002（石畳案）
18区画の麻の葉を **「線＝御影石ピンコロの石帯 ／ 面＝乱張り石」の石畳**で構成する。麻の葉の「線（18区画境界）」と「区画の中身（充填）」を分離し、線も面も自然石で和の重厚感を採る。
- **線（縁取り）**: 御影石ピンコロ 60〜90mm 角（または同幅平板）を区画境界に沿って並べる。溶接不要・目地調整で取り合い誤差を吸収。
- **面（充填）**: 乱張り石（鉄平石・錆系自然石など）。色は石種で決まる。
- **下地**: 砕石路盤 → コンクリート下地 → モルタル張り。排水は緩勾配（外周芝生へ）または砂利目地透水。
- **火床（中心⌀1000）**: 薪直火。内周を耐火煉瓦で巻き、天端から1〜2段立ち上げて灰止めリム＋砂利排水底。
- **却下/対案**: A 琉球石灰岩 切り出し石（コスト高）/ B レンガ敷き（自前目地が麻の葉線と競合）/ コールテン案（線＝コールテン鋼見切り＋面＝豆砂利洗い出し。意匠の鮮度・面一純度は上だが、和の重厚感で石畳を採る）。理由は ADR-0002。

## スコープ
- **単一の成果物**（焚き火台1基）を、複数の図面シートで一意に定義する。
  - A-01 平面図 PLAN（意匠割付・全体寸法）
  - A-02 A-A 断面図 SECTION（埋め込み・下地・火床の鉛直構成）
  - A-03 構成詳細図 … ピンコロ石帯の割付＋乱張りの石割り図（→ ADR-0002・石畳案）。
- **構成材の製作/調達 ＋ 現場設置/造作の両方**をカバーする。

- 構造計算・申請図書（意匠・寸法の製作/設置指示にとどめる）。

## 読み手
造園/外構業者（構成材の加工/調達と現場設置を担う）。図面一式でファイヤーピットが一意に製作・設置できる必要がある（→ 成功基準）。

## 成功基準
業者がこの図面一式だけで製作見積・製作・設置できること。具体的には:
1. 形状が一意に決まる情報: 外形寸法 / 18区画の割付寸法・角度 / 焚き火部寸法 / 構成材の材質・寸法
2. 設置が一意に決まる情報: 埋め込み・下地構成 / 火床の鉛直構成 / GL基準（天端＝芝生面一）
3. 印刷して手渡せる体裁: A4 landscape / 縮尺明記 / 表題欄・凡例・注記

※ 構成要素（材料・工法）は ADR-0002（石畳案）で確定済み。1・2 の値は A-02/A-03 でこの決定に従って作図する。

## 再現性（このリポジトリの存在理由）
旧成果物（`reference/legacy/`）は **土台を決めずに一発で作られたため再現性がない**。本リポジトリは土台（docs ＋ パラメトリックな作図ソース）から起こし直し、**同じ意匠を、構成の選択肢を差し替えながら**再現可能に作れる状態にする。
