# DESIGN — 意匠と図面の見せ方

「どんな意匠を、どう見せるか」を集約する。**意匠の幾何も構成要素も確定**（構成 → ADR-0002・芝生目地案）。

## 意匠（確定・幾何）

### 麻の葉文様 ファイヤーピット
- 外形: 正六角形（point-up）。全高 6000mm = 2R、R（中心〜頂点）= 3000、全幅（対辺フラット間）= √3×R ≈ 5196mm、一辺 = R = 3000mm。
- 割付: 中心から6つの正三角形（セクター S1〜S6）へ分割 → 各セクターを**重心スポーク**で3区画（A/B/C）に分割 → 計18区画。
- 区画形状: 二等辺三角形 約 3000 × 1732 × 1732mm（重心＝正三角形外接円中心、外接円半径 = 辺/√3 ≈ 1732）。
- 中央: 焚き火部 ⌀1000（小六角形、半径500）をくり抜く。
- 対称: 全寸法 mm。中心線より対称配置。

#### 頂点・割付の生成規則（パラメトリックの種）
- 六角形頂点 V1〜V6: `angle = 90° + i·60°`（i=0..5、point-up）、`(R·cosθ, −R·sinθ)`。
- セクター i: 三角形 (中心, Vi, Vi+1)。重心 `g = (中心+Vi+Vi+1)/3`。
- 区画: A=(中心,Vi,g) / B=(Vi,Vi+1,g) / C=(Vi+1,中心,g)。
- 焚き火部小六角形: 同じ角度規則で半径500。
- ※ 確定値は R=3000、焚き火半径=500。これらを唯一の入力（パラメータ）として全図形が決まる（→ ARCHITECTURE「C1」）。

## 構成（確定 — ADR-0002・芝生目地案）
麻の葉の線を芝生（負の空間）で出し、区画を舗装の島とする。見せ方は以下で確定。
- **線**: 幅 ≈120mm の芝生帯を18区画境界に通し、外周の芝生と連続させる（島が芝生に浮かぶ）。外形に硬い見切り線を作らない。
- **面**: 豆砂利洗い出し・単色の区画島。緑と対比し縁がシャープで麻の葉が読める。
- **中心**: 芝生帯は火床の耐火煉瓦リムまで通す（焚き火部の六角開口が収束点を抜くため潰れず、12本がリムに取り付く）。
- **下地・鉛直（A-02 断面の中身）**: 区画島＝砕石路盤＋透水路盤＋透水洗い出し。芝生帯＝客土＋芝（島の縁にエッジ材）。全面 GL±0・全面浸透。中心火床は耐火煉瓦巻き＋灰止めリム＋砂利排水底。
- **A-03 の性格**: 区画島の割付＋芝生帯（幅・エッジ材）取り合い＋火床カラーの納まり。

## 図面シート（構成）
A4 landscape の図面シート形式。全3シート。
- **A-01 平面図 PLAN** … 意匠割付・全体寸法・焚き火部・凡例・北マーク・注記。意匠確定分で先行可能。
- **A-02 A-A 断面図 SECTION** … V1–V4 を通る対角断面（全長6000）。埋め込み・砕石路盤・透水洗い出し島・芝生帯（客土）・エッジ材・中心火床（耐火煉瓦巻き＋灰止めリム＋砂利排水底）の鉛直構成（→ ADR-0002・芝生目地案）。
- **A-03 構成詳細図** … 区画島の割付＋芝生帯の幅・エッジ材取り合い＋芝生帯と火床リムの納まり（→ ADR-0002・芝生目地案）。

## 見せ方（共通レイアウト方針）
JIS 製図に倣った図面シート形式。コピー元は `template/template.html`。
- 用紙: A4 landscape。印刷して手渡せること。
- 図面枠: 二重枠＋センタリングマーク。
- 構成要素: 図面本体（平面/断面/詳細）＋ 表題欄 ＋ 凡例 ＋ 注記。
- 表題欄: PROJECT「麻の葉 FIRE PIT」/ シート番号（A-01〜）/ 縮尺 / 単位mm / 作図日。
- 必須表記: 寸法線 / 縮尺 / 表題欄の各値 / 注記。
- 配色: 紙白（#f5f1e8）＋墨（#1a1a1a）。画面は極薄方眼、印刷は白地で崩れないこと。
- 縮尺の目安（旧成果物準拠、A4化で再調整）: A-01 S=1:50 / A-02 S=1:30 / A-03 S=1:50・1:25。

## 注記（確定 — ADR-0002・芝生目地案）
1. 全寸法 mm 単位。中心線より対称配置。
2. 麻の葉文様 = 6セクター × 3分割 = 計18区画。
3. 線は芝生帯（幅≈120mm・外周芝生と連続）、区画島は豆砂利洗い出し単色。島の縁にエッジ材で縁切り。全面浸透（水勾配なし）。
4. 中心⌀1000 は薪直火の火床。耐火煉瓦巻き＋灰止めリム（天端から1〜2段立ち上げ）＋砂利排水底。芝生帯は耐火煉瓦リムまで通す（最内の芝は傷んだら差し替え）。
5. 天端と周囲芝生面を面一（GL±0）。ただし中心火床のみ立ち上がり例外。
