# ADR-0002 — 構成要素を「コールテン鋼見切り（線）＋豆砂利洗い出し（面）」に決定

## 文脈
18区画の麻の葉意匠（幾何は確定）を「何で・どう」物理的に構成するかが未決定だった（BRIEF「検討中」）。これが確定しないと A-02 断面・A-03 構成詳細を起こせない。旧成果物（`reference/legacy/`）の **A案＝琉球石灰岩の切り出し石（t200 / 18石切り出し / 目地100mm 砂利充填）はコスト高で却下済み**。残る候補として B レンガ敷き / C 石畳が挙がっていた。

評価軸は コスト / 意匠再現性 / 施工性 / 耐火・耐候。本案件は宿の屋外（和の意匠）、天端は周囲芝生と面一（GL±0）、中心 ⌀1000 は薪直火の焚き火部。

## 決定
構成を **「線＝コールテン鋼見切り材 ＋ 面＝豆砂利洗い出し（単色）」のハイブリッド**とする。B/C のような単一材で全体を作る発想を採らず、**麻の葉の「線（18区画境界＝放射状スポーク）」と「区画の中身（充填）」を分離して別々に決める**。

確定値:

- **線（縁取り／見切り材）**: **耐候性鋼（コールテン鋼）見切り**。t6〜9 × 露出高さ 100〜150mm 程度を連続して立てる。18区画境界・外周・中心リムの取り合いに用いる。錆安定後は塗装不要・保守軽微、熱に強い。錆色が豆砂利洗い出しと調和する。
- **面（充填）**: **豆砂利洗い出し・単色**（透水モルタル仕上げ）。面が静かで線が際立つ。18区画すべて同一砂利＝調達・施工が単純でコスト最低。
- **下地（A-02 鉛直構成）**: 上から ①透水性洗い出し仕上げ → ②透水性路盤（ポーラスコンクリート／透水モルタル）→ ③砕石浸透路盤（転圧）→ 路床。**水勾配なしのフラット**とし、雨水は面を抜けて**全面浸透**で処理（→ 後述「面一と排水の矛盾」）。
- **火床（中心 ⌀1000）**: 薪直火。くり抜き内周を**耐火煉瓦で巻き**、天端から **1〜2段（≈65〜130mm）立ち上げて灰止めリム**にする（灰が外周へ飛ばない）。底に割栗石＋砂利の排水・浸透層。清掃は手作業。
- **納まり**: 外周の区画／芝生境界は GL±0 面一（コールテン見切りが土留め兼用）。**中心火床のみ立ち上がりを設ける例外**。
- **A-03 の性格**: 切り出しパターン図ではなく、**コールテン見切りの割付・取り合い図（線取り合い図）**になる。充填が単色洗い出しのため区画内の割れ付け（部材分割）は不要で、見切り材の定尺・コーナー・中心リム取り合いが主題。

## 理由
- **意匠再現性 × コストの両立**: 麻の葉の本体は「線」。A案が高かったのは線を「部材そのものの継ぎ目」で出すために18枚を三角に切り出したから。線を見切り材に分離すれば、面は安い洗い出しでよく、A案のコスト構造を回避できる。
- **線の鮮度**: 充填にレンガ（B）を使うと自前の目地グリッドが麻の葉線と視覚的に競合し、端部に切りも出る。穏やかな洗い出し単色のほうが線が際立つ。コールテン見切りは連続・薄く、鋭い線を出せる。
- **耐候・耐火**: コールテン鋼は屋外で安定錆、塗装不要。中心へ収束する線が熱を受けても金属は耐える。火床は耐火煉瓦＋立ち上がりリムで灰と熱を封じる。
- **面一の純度**: 透水仕上げ＋全面浸透なら水勾配が不要で、5m幅の面を完全フラットに保てる。麻の葉の対称が勾配で歪まない。
- **差し替え容易性（ADR-0001 の拡張）**: 縁取り材種・部材寸法・充填材を `lib/firepit.js`（または単一定数ブロック）に名前付きパラメータとして置く。意匠の幾何（A-01）は材料非依存で不変、構成変更の影響は構成依存の A-02/A-03 に局所化される。BRIEF「構成の選択肢を差し替えながら再現可能」を構成決定の段階で仕込む。

## トレードオフ / 却下案
- **A 琉球石灰岩 切り出し石**: コスト高で却下（既定）。18石切り出し＋t200 が主因。
- **B レンガ敷き（単一材）**: 施工性・コスト予見性は高いが、レンガ自前の目地グリッドが麻の葉線と競合し、三角区画端部に切りが多発。線の鮮度で劣り却下。なお「線＝コールテン／面の充填材」としてのレンガは将来の差し替え候補に残せる。
- **C 石畳（乱張り）**: 高級感・耐久は最高だが、材料・施工費がA案帯に戻り職人依存。コスト軸で却下。
- **排水を外周芝生へ緩勾配 / 中心火床へ集水**: いずれも面一の純度を崩す（対称が傾く）か、火を焚く中心に水が集まる難点。透水＋全面浸透を採用。
- **火床のガスバーナー化**: 配管・バーナー・ガス源で別設計＝スコープ拡大。本案件の前提（薪直火）から外れるため対象外。
- **可搬火鉢の落とし込み**: 清掃は最楽だが既製品選定が要り（custom 製作前提に反する）、面一の意匠が損なわれるため不採用。
