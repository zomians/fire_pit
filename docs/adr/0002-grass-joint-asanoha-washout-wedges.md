# ADR-0002 — 構成要素を「芝生目地（線）＋豆砂利洗い出し区画（面）」に決定

> ⚠️ 本 ADR は **芝生目地案** の候補版。コールテン案（`0002-corten-edging-washout-aggregate-fill.md` / PR #4）・石畳案（`0002-stone-paving-rubble-with-pinkoro-edging.md` / PR #5）と同じ分岐点（main）から起こした対の候補で、**採用された1案だけが ADR-0002 として main に残る**（他は PR ごと close）。

## 文脈
18区画の麻の葉意匠（幾何は確定）を物理的にどう構成するかを決める。A案＝琉球石灰岩 切り出し石はコスト却下済み。施工はユーザー本人（造園/外構業者）。施工性は業者前提で評価する。

本案は「麻の葉の線」をモノ（金属・石）で出すのではなく、**芝生（負の空間）で出す**点が他案と根本的に違う。18区画を舗装の島とし、その境界（麻の葉の線）を芝生の帯とする。**この芝生は外周の既存芝生と連続する**（島々が芝生の海に浮かぶ構図）。天端は全面 GL±0 面一。

## 決定
構成を **「線＝芝生の帯（外周芝生と連続） ＋ 面＝豆砂利洗い出し（単色）の区画島」**とする。

確定値:

- **線（区画境界）= 芝生の帯**。麻の葉の18区画境界を、幅 **≈120mm の芝生の目地**とする。外周の芝生と連続し、内部の芝生網が境界で外へ流れ出る（外形に硬い見切り線を作らず、区画島の外縁が芝生に溶ける）。
- **面（区画）= 豆砂利洗い出し・単色**（透水モルタル仕上げ）の島。緑の芝生と対比し、島の縁がシャープで麻の葉が読める。18島すべて同色。
- **中心の納まり**: 18区画は中心の焚き火部へ収束するため、芝生の帯は中心へ行くほど細る。**芝生は中心まで伸ばさず、火床周りは舗装のカラー（襟）で受ける**（火床⌀1000 の外側に半径 ≈900 程度の洗い出しカラー）。**芝生が火に触れない**＝耐火・焦げ防止の面でも合理的。
- **下地（A-02 鉛直構成）**: 区画島は 砕石路盤 → 透水路盤 → 透水性洗い出し。芝生帯は 客土＋芝（島の縁にエッジ材＝土留めで舗装と縁切り）。全面 GL±0 面一。排水は島が透水＋芝生帯の浸透で全面浸透（水勾配なし）。
- **火床（中心 ⌀1000）**: 薪直火。内周を耐火煉瓦で巻き、天端から1〜2段（≈65〜130mm）立ち上げて灰止めリム。底に割栗石＋砂利の排水・浸透層。
- **納まり**: 全面 GL±0 面一（芝生帯・舗装島・外周芝生が同レベルで連続）。中心火床のみ立ち上がり例外。
- **A-03 の性格**: 区画島の割付図＋芝生帯（幅・エッジ材）の取り合い図＋火床カラーの納まり。

## 理由
- **面一・連続の純度が最高**: 芝生帯が外周芝生と同素材・同レベルで繋がるため、面一の納まりが最も素直で、焚き火台が庭に溶け込む。
- **意匠が軽やか・自然**: 緑の線が麻の葉を描き、舗装島が浮かぶ。和モダンより自然・庭的な表情。
- **耐火的にも整合**: 火床周りを舗装カラーで受け芝生を遠ざけるルールが、焦げ防止と中心収束問題を同時に解く。
- **線材コストが不要**: 麻の葉の線にコールテン鋼・石帯のような部材費がかからない（その分、芝生帯の造成・維持コストに置き換わる）。

## トレードオフ / 却下案
- **芝生帯の維持管理**: 幅120mm の芝の帯18本＋分岐の刈り込み・際の処理は手間（業者なら可だが、コールテン/石畳のノーメンテに対し継続管理が要る）。踏圧で擦り切れる懸念（動線次第）。
- **線の鮮度**: 芝生帯は幅があり縁が有機的なので、コールテン鋼の鋭い線ほどシャープではない。自然な表情と引き換え。
- **中心の制約**: 芝生を火床まで伸ばせないため、中心は舗装カラーになる（純粋な「芝生だけの線」が中心では途切れる）。
- **A 琉球石灰岩 切り出し石**: コスト却下（既定）。
- **コールテン案（PR #4）/ 石畳案（PR #5）**: 線をモノで出す対案。線の鮮度（コールテン）や重厚感（石畳）で勝るが、本案は**庭との連続・面一の素直さ・自然な表情**で選好。
- **B レンガ敷き**: 自前目地が麻の葉線と競合し却下。
