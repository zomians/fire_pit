# ADR-0002 — 構成要素を「芝生目地（線）＋乱張り石区画（面）」に決定

> ⚠️ 本 ADR は **芝生目地案** の候補版。コールテン案（`0002-corten-edging-washout-aggregate-fill.md` / PR #4）・石畳案（`0002-stone-paving-rubble-with-pinkoro-edging.md` / PR #5）と同じ分岐点（main）から起こした対の候補で、**採用された1案だけが ADR-0002 として main に残る**（他は PR ごと close）。

## 文脈
18区画の麻の葉意匠（幾何は確定）を物理的にどう構成するかを決める。A案＝琉球石灰岩 切り出し石はコスト却下済み。施工はユーザー本人（造園/外構業者）。施工性は業者前提で評価する。

本案は「麻の葉の線」をモノ（金属・石帯）で出すのではなく、**芝生（負の空間）で出す**点が他案と根本的に違う。18区画を**乱張り石の島**とし、その境界（麻の葉の線）を芝生の帯とする。**この芝生は外周の既存芝生と連続する**（乱張り石の島々が芝生の海に浮かぶ構図）。天端は全面 GL±0 面一。

## 決定
構成を **「線＝芝生の帯（外周芝生と連続） ＋ 面＝乱張り石の区画島」**とする。

確定値:

- **線（区画境界）= 芝生の帯**。麻の葉の18区画境界を、幅 **≈120mm の芝生の目地**とする。外周の芝生と連続し、内部の芝生網が境界で外へ流れ出る（外形に硬い見切り線を作らず、区画島の外縁が芝生に溶ける）。
- **面（区画）= 乱張り石の島**。各区画を**大判の不定形な平石（乱張り）**で張り、はっきりした目地が走る。**砂利・洗い出し・小粒の敷石ではない**。緑の芝生と対比し、石の島が浮かぶ。色は石種で決まる（落ち着いた灰〜錆茶系）。
- **中心の納まり**: 18区画は中心の焚き火部へ収束するが、中心は焚き火部（⌀1000 の六角形）で抜けるため芝生帯は一点に潰れない。**芝生帯は火床の耐火煉瓦リムまで通す**（リム外周で12本の芝生帯が約30°間隔で取り付く。舗装カラー＝襟は設けない）。立ち上げた耐火煉瓦リムが熱の緩衝になる。最内の芝が熱で傷んだら差し替える前提（焦げリスクは受容）。
- **下地（A-02 鉛直構成）**: 区画島は 砕石路盤 → コンクリート下地 → モルタル張りで乱張り石を据える。芝生帯は 客土＋芝（島の縁にエッジ材＝土留めで石島と縁切り）。全面 GL±0 面一。排水は雨水が石島から芝生帯へ流れ、芝生目地で全面浸透（水勾配なし）。
- **火床（中心 ⌀1000）**: 薪直火。内周を耐火煉瓦で巻き、天端から1〜2段（≈65〜130mm）立ち上げて灰止めリム。底に割栗石＋砂利の排水・浸透層。
- **納まり**: 全面 GL±0 面一（芝生帯・乱張り石島・外周芝生が同レベルで連続）。中心火床のみ立ち上がり例外。
- **A-03 の性格**: 乱張り石島の割付・石割り＋芝生帯（幅・エッジ材）の取り合い＋芝生帯と火床リムの納まり。

## 理由
- **面一・連続の純度が最高**: 芝生帯が外周芝生と同素材・同レベルで繋がるため、面一の納まりが最も素直で、焚き火台が庭に溶け込む。
- **意匠が自然・庭的**: 緑の線が麻の葉を描き、乱張り石の島が浮かぶ。石の重厚感と芝の柔らかさを両立し、和庭の定番（乱張り＋芝）の表情。
- **線が中心まで完結**: 芝生帯を耐火煉瓦リムまで通すことで、麻の葉の線が中心の火まで途切れず届く。焚き火部の六角開口が収束点を抜くため、線が潰れず12本がリムに取り付く。
- **排水が素直**: 乱張り石島は不透水だが、雨は芝生目地へ流れて浸透するため、面一のまま水勾配が要らない。

## トレードオフ / 却下案
- **芝生帯の維持管理**: 幅120mm の芝の帯18本＋分岐の刈り込み・際の処理は手間（業者なら可だが、コールテン/石畳のノーメンテに対し継続管理が要る）。踏圧で擦り切れる懸念（動線次第）。
- **線の鮮度**: 芝生帯は幅があり縁が有機的なので、コールテン鋼の鋭い線ほどシャープではない。自然な表情と引き換え。
- **最内の芝の焦げ**: 芝生帯を耐火煉瓦リムまで通すため、最内の芝はリムの熱を受ける。立ち上げリムが緩衝になるが、傷んだら差し替える前提（舗装カラーで遠ざける案は意匠を優先して不採用）。
- **A 琉球石灰岩 切り出し石**: コスト却下（既定）。
- **コールテン案（PR #4）/ 石畳案（PR #5）**: 線をモノで出す対案。線の鮮度（コールテン）や、石帯で締まった構え（石畳）で勝るが、本案は**庭との連続・面一の素直さ・乱張り石が芝に浮かぶ自然な表情**で選好。石畳案と面材（乱張り石）は同じで、違いは線（芝生 vs 石帯）。
- **B レンガ敷き**: 自前目地が麻の葉線と競合し却下。
