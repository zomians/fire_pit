# ADR-0002 — 構成要素を「石帯ピンコロ（線）＋乱張り石（面）」の石畳に決定

> ⚠️ 本 ADR は **石畳案（C系）** の候補版。コールテン案（`0002-corten-edging-washout-aggregate-fill.md`）と同じ分岐点（main）から起こした対の候補で、**採用された片方だけが ADR-0002 として main に残る**（他方の PR は close）。

## 文脈
18区画の麻の葉意匠（幾何は確定）を物理的にどう構成するかを決める。旧成果物の **A案＝琉球石灰岩の切り出し石はコスト高で却下済み**。残る候補は B レンガ敷き / C 石畳。評価軸は コスト / 意匠再現性 / 施工性 / 耐火・耐候。

施工は **ユーザー本人（造園/外構業者）が行う**。したがって施工性は「業者の技能・道具・外注ルート前提」で評価する（金属加工・乱張り・洗い出し・耐火煉瓦はいずれも本人の土俵）。コールテン案との違いは施工可否ではなく **意匠の質（和の重厚感）と納まり（面一・排水）** の選好で分かれる。

本案件は宿の屋外（和の意匠）、天端は周囲芝生と面一（GL±0）、中心 ⌀1000 は薪直火の焚き火部。

## 決定
構成を **「線＝御影石ピンコロ（または平板）の石帯 ＋ 面＝乱張り石」の石畳**とする。麻の葉の「線（18区画境界）」と「区画の中身（充填）」を分離して決める点はコールテン案と同じだが、**線も面も自然石**で構成し、和の重厚感を採る。

確定値:

- **線（縁取り／石帯）**: **御影石ピンコロ 60〜90mm 角（または同幅の平板）を区画境界に沿って並べる**。18区画境界・外周・中心リム取り合いに用いる。溶接・精密金属加工が不要で、目地調整により取り合いの誤差を吸収できる。線幅は太め（石の存在感のある線）。
- **面（充填）**: **乱張り石**（鉄平石・錆系自然石など）。各区画を自然石の乱張りで埋める。色味は石種で決まる（自然な色幅を許容）。
- **下地（A-02 鉛直構成）**: 砕石路盤（転圧）→ コンクリート下地 → モルタル張りで石を据える。乱張りはモルタル張りが標準のため、**排水は緩い水勾配（外周芝生へ）または砂利目地での透水**で処理する。面一は外周で担保するが、コールテン案の「水勾配ゼロの透水フラット」ほど厳密にはしにくい（→ トレードオフ）。
- **火床（中心 ⌀1000）**: 薪直火。くり抜き内周を**耐火煉瓦で巻き**、天端から **1〜2段（≈65〜130mm）立ち上げて灰止めリム**にする。底に割栗石＋砂利の排水・浸透層。
- **納まり**: 外周の区画／芝生境界は GL±0 面一（石端＋見切り）。**中心火床のみ立ち上がりを設ける例外**。
- **A-03 の性格**: ピンコロ石帯の割付図＋乱張りの石割り（部材分割の目安）。

## 理由
- **和の重厚感・本格感**: 自然石の乱張り＋石帯は、宿の屋外に石庭的な格と重みを与える。鋼の線＋洗い出しの軽快さより、石の質感を採る選好。
- **線の出し方が組積で完結**: 線を石帯にすることで金属の溶接・精密切断が不要になり、目地で取り合い誤差を吸収できる。業者の組積技能の土俵で完結する。
- **耐久・メンテ**: 自然石は半永久・ほぼノーメンテ。屋外・薪の熱環境にも強い。
- **意匠再現性のトレードオフは受容**: 石帯の線は太く、乱張り面に自然な目地が出るため、麻の葉線の鮮度はコールテン案に劣る。ここは和の質感と引き換えに受け入れる。

## トレードオフ / 却下案
- **A 琉球石灰岩 切り出し石**: コスト高で却下（既定）。18石切り出し＋t200 が主因。
- **コールテン案（線＝コールテン鋼見切り／面＝豆砂利洗い出し）**: 線が細く鮮明で麻の葉の鮮度が高く、透水フラットで面一の純度も高い。本案（石畳）は意匠再現性・面一純度でこれに劣るが、**和の重厚感**で上回ると判断。コールテン案は対の候補 ADR として並走（`0002-corten-edging-washout-aggregate-fill.md` / PR は別）。
- **B レンガ敷き（単一材）**: レンガ自前の目地グリッドが麻の葉線と競合し端部切りが多発。和の質感でも自然石に劣り却下。
- **排水を全面透水フラットに**: 乱張りモルタル張りでは成立しにくい。緩勾配／砂利目地透水で対応。
